== Description ==

Fragmentions for WordPress is a plugin to add Jonathan Neal's [Fragmentions script](https://github.com/chapmanu/fragmentions) to WordPress.

== Installation ==

1. Upload the `fragmention` directory to your site's `plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress

== Frequently Asked Questions ==

= Are there any plugin options? =
Nope. Just activate the plugin and go. No configuration needed.
